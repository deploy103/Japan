# 일본어 문장 분석 기반 학습 보조 웹 플랫폼

---

# 프로젝트 소개

이 프로젝트는 일본어 학습자가 문장을 더 쉽게 이해하고 학습할 수 있도록 돕는 웹 기반 일본어 학습 보조 프로그램이다.

단순히 일본어 문장을 한국어로 번역하는 기능에서 끝나지 않고, 일본어 문장 속 한자 추출, 후리가나 표시, 단어 분리, 품사 분석, 음성 인식, 이미지 OCR 등의 기능을 제공한다.

일본어는 히라가나, 가타카나, 한자가 함께 사용되며, 특히 한자의 읽는 법과 의미를 파악하는 것이 학습자에게 큰 어려움이 된다. 본 프로젝트는 이러한 문제를 해결하기 위해 문장 분석과 학습 보조 기능을 통합하였다.

또한 PC와 모바일 환경 모두에서 사용할 수 있도록 반응형 웹으로 구현하여, 사용자가 장소와 기기에 관계없이 일본어 학습을 이어갈 수 있도록 설계하였다.

---

# 제작 동기

일본어를 학습하면서 가장 어렵게 느껴지는 부분은 문장 속 한자의 읽는 법과 의미를 파악하는 것이었다. 기존 번역기는 문장 전체의 의미는 알려주지만, 어떤 단어가 어떤 역할을 하는지, 한자를 어떻게 읽는지, 문장 구조가 어떻게 이루어져 있는지까지 자세히 알려주지는 않는다.

그래서 본 프로젝트에서는 단순 번역 기능을 넘어 일본어 문장을 학습 관점에서 분석할 수 있는 웹 프로그램을 만들고자 하였다. 사용자는 일본어 문장을 입력하거나 이미지, 음성으로 입력할 수 있으며, 프로그램은 번역 결과와 함께 한자, 후리가나, 단어 분석, 품사 정보 등을 제공한다.

이를 통해 사용자는 일본어 문장의 의미뿐만 아니라 문자 체계, 발음, 문법 구조까지 함께 학습할 수 있다.

---

# 프로젝트 목표

- 일본어 문장을 한국어로 번역할 수 있도록 한다.
- 일본어 문장 속 한자를 쉽게 분석할 수 있도록 한다.
- 후리가나를 통해 한자의 읽는 법을 제공한다.
- 일본어 문장을 단어 단위로 분석할 수 있도록 한다.
- 음성 및 이미지 기반 입력 기능을 지원한다.
- 모바일과 PC 환경 모두에서 사용할 수 있도록 한다.
- 단순 번역기가 아닌 일본어 학습 보조 플랫폼을 만드는 것을 목표로 한다.

---

# 주요 기능

## 1. 텍스트 번역 기능

사용자가 일본어 문장을 직접 입력하면 한국어로 번역해주는 기능이다.

예시:

```txt
私は昨日、図書館で日本語の本を読みました。
```

출력:

```txt
나는 어제 도서관에서 일본어 책을 읽었습니다.
```

### 기능 의미

이 기능은 일본어 문장을 한국어로 변환하여 사용자가 문장의 전체적인 의미를 빠르게 파악할 수 있도록 돕는다. 단순 번역 기능에서 끝나는 것이 아니라, 번역 결과를 바탕으로 단어 분석, 한자 검색, 후리가나 표시 기능과 연동하여 일본어 학습 효율을 높이는 것을 목표로 한다.

---

## 2. 한국어 → 일본어 번역 기능

한국어 문장을 일본어로 번역하는 기능이다.

예시:

```txt
나는 일본어를 공부하고 있습니다.
```

출력:

```txt
私は日本語を勉強しています。
```

### 기능 의미

한국어 문장을 일본어로 변환하는 기능을 통해 사용자가 일본어 문장 구성 방식을 익힐 수 있도록 하였다. 이를 통해 단순히 번역 결과를 확인하는 것이 아니라, 한국어와 일본어의 어순 차이, 조사 사용, 표현 방식의 차이를 비교하며 학습할 수 있다.

---

## 3. 음성 인식 기능

사용자가 일본어를 말하면 음성을 텍스트로 변환하는 기능이다.

예시:

```txt
こんにちは、私は学生です。
```

### 기능 의미

음성 인식 기능은 사용자가 직접 일본어를 말하고, 그 발화가 어떤 텍스트로 인식되는지 확인할 수 있도록 한다. 이를 통해 일본어 발음 연습과 듣기·말하기 학습을 동시에 지원할 수 있다.

---

## 4. 일본어 음성 출력 기능

일본어 문장을 실제 발음으로 읽어주는 기능이다.

### 기능 의미

음성 출력 기능은 일본어 문장을 실제 발음으로 들을 수 있게 하여, 사용자가 발음과 억양을 함께 학습할 수 있도록 한다. 이는 문자 중심의 학습에서 벗어나 듣기와 말하기 능력 향상에도 도움을 준다.

---

## 5. 한자 자동 추출 기능

일본어 문장에서 한자만 따로 추출하는 기능이다.

예시:

```txt
私は図書館で日本語を勉強します。
```

출력:

```txt
私
図
書
館
日
本
語
勉
強
```

### 기능 의미

일본어 문장에는 히라가나, 가타카나, 한자가 섞여 있어 초보 학습자가 문장을 이해하는 데 어려움을 느낀다. 한자 자동 추출 기능은 문장 속 한자만 분리하여 보여줌으로써 사용자가 모르는 한자를 빠르게 파악하고 학습할 수 있도록 돕는다.

---

## 6. 한자 상세 검색 기능

추출된 한자를 클릭하면 뜻, 음독, 훈독, 예시 단어를 보여주는 기능이다.

예시:

```txt
学
```

출력:

```txt
뜻: 배우다
음독: ガク
훈독: まなぶ
예시: 学校, 学生, 学ぶ
```

### 기능 의미

한자는 같은 글자라도 여러 읽기 방식과 의미를 가질 수 있기 때문에 단순 번역만으로는 학습에 한계가 있다. 한자 상세 검색 기능은 한자의 뜻, 음독, 훈독, 예시 단어를 제공하여 사용자가 한자를 체계적으로 이해할 수 있도록 한다.

---

## 7. 후리가나 표시 기능

한자 위나 옆에 읽는 법을 표시하는 기능이다.

예시:

```txt
日本語(にほんご)
```

### 기능 의미

후리가나는 일본어 한자의 읽는 법을 알려주는 표기 방식이다. 본 프로그램은 한자가 포함된 문장에 후리가나를 표시하여 사용자가 문장을 더 쉽게 읽고 발음할 수 있도록 하였다.

---

## 8. 단어 분리 기능

일본어 문장을 단어 단위로 분리하는 기능이다.

예시:

```txt
私は昨日図書館で勉強しました。
```

출력:

```txt
私 / は / 昨日 / 図書館 / で / 勉強 / しました
```

### 기능 의미

일본어는 한국어와 달리 문장에 띄어쓰기가 거의 없기 때문에 초보 학습자가 단어 경계를 파악하기 어렵다. 단어 분리 기능은 문장을 형태소 단위로 나누어 보여줌으로써 문장 구조를 쉽게 이해할 수 있도록 한다.

---

## 9. 품사 표시 기능

단어별 품사를 표시하는 기능이다.

예시:

```txt
私: 명사
は: 조사
昨日: 명사
図書館: 명사
で: 조사
勉強しました: 동사
```

### 기능 의미

품사 표시 기능은 일본어 문장을 구성하는 각 단어의 역할을 보여준다. 이를 통해 사용자는 조사, 명사, 동사 등의 문법 요소를 구분하며 일본어 문장 구조를 더 명확하게 이해할 수 있다.

---

## 10. 조사 설명 기능

문장 속 조사를 분석하고 설명하는 기능이다.

예시:

```txt
私は学校で勉強します。
```

출력:

```txt
は: 주제를 나타내는 조사
で: 장소를 나타내는 조사
```

### 기능 의미

일본어의 조사는 문장의 의미와 구조를 결정하는 중요한 요소이다. 본 프로그램은 문장 속 조사를 자동으로 감지하고 각각의 역할을 설명하여 사용자가 일본어 문법을 더 쉽게 이해할 수 있도록 하였다.

---

## 11. 이미지 OCR 기능

이미지 속 일본어 문자를 인식하여 텍스트로 변환하는 기능이다.

### 기능 의미

OCR 기능은 이미지 속 일본어 문자를 인식하여 텍스트로 변환하는 기능이다. 이를 통해 사용자는 교과서, 학습지, 일본어 간판, 이미지 자료 등에 포함된 일본어를 직접 입력하지 않고도 분석할 수 있다.

---

## 12. 카메라 인식 기능

모바일 카메라로 일본어 문장을 촬영하여 분석하는 기능이다.

### 기능 의미

모바일 환경에서는 사용자가 일본어 문장을 직접 입력하기보다 카메라로 촬영하는 상황이 많다. 카메라 인식 기능은 모바일 기기의 특성을 활용하여 이미지 속 일본어를 빠르게 인식하고 번역할 수 있도록 설계하였다.

---

## 13. 가타카나 단어 분석 기능

가타카나 단어를 추출하고 원래 의미를 분석하는 기능이다.

예시:

```txt
コンピューター → 컴퓨터
ホテル → 호텔
```

### 기능 의미

가타카나는 주로 외래어를 표기할 때 사용되므로, 일본어 학습자가 단어의 의미를 추측하는 데 중요한 단서가 된다. 본 프로그램은 가타카나 단어를 따로 추출하여 사용자가 외래어 표현을 쉽게 이해할 수 있도록 하였다.

---

## 14. 히라가나/가타카나 변환 기능

히라가나와 가타카나를 서로 변환하는 기능이다.

예시:

```txt
こんにちは → コンニチハ
コンピューター → こんぴゅーたー
```

### 기능 의미

일본어는 히라가나, 가타카나, 한자라는 여러 문자 체계를 사용한다. 문자 변환 기능은 히라가나와 가타카나의 대응 관계를 확인할 수 있게 하여 일본어 문자 학습에 도움을 준다.

---

## 15. JLPT 난이도 표시 기능

단어나 한자의 JLPT 난이도를 표시하는 기능이다.

예시:

```txt
学校: N5
経済: N3
```

### 기능 의미

JLPT 난이도 표시 기능은 단어와 한자의 학습 수준을 구분하여 보여준다. 이를 통해 사용자는 자신의 수준에 맞는 단어를 우선적으로 학습할 수 있으며, 체계적인 일본어 학습 계획을 세울 수 있다.

---

## 16. 단어장 저장 기능

모르는 단어를 저장하는 기능이다.

### 기능 의미

단어장 저장 기능은 사용자가 모르는 단어나 자주 틀리는 표현을 따로 저장할 수 있도록 한다. 이를 통해 일회성 번역에서 끝나지 않고 반복 학습이 가능하도록 하였다.

---

## 17. 복습 퀴즈 기능

저장한 단어를 기반으로 퀴즈를 생성하는 기능이다.

### 기능 의미

복습 퀴즈 기능은 사용자가 저장한 단어를 바탕으로 문제를 생성하여 반복 학습을 유도한다. 이를 통해 단순히 단어를 저장하는 데 그치지 않고, 사용자가 기억 여부를 스스로 확인할 수 있도록 하였다.

---

## 18. 오답노트 기능

퀴즈에서 틀린 단어를 따로 저장하는 기능이다.

### 기능 의미

오답노트 기능은 사용자가 퀴즈에서 틀린 단어를 자동으로 저장하여 취약한 부분을 확인할 수 있도록 한다. 이를 통해 개인별 학습 약점을 분석하고 반복 학습할 수 있다.

---

## 19. 문장 난이도 분석 기능

문장의 난이도를 분석하는 기능이다.

### 기능 의미

문장 난이도 분석 기능은 문장에 포함된 한자 비율, 단어 수준, 문장 길이 등을 기준으로 학습 난이도를 추정한다. 이를 통해 사용자는 자신의 수준에 맞는 문장을 선택하여 학습할 수 있다.

---

## 20. 문장 구조 분석 기능

문장을 주어, 목적어, 동사 중심으로 분석하는 기능이다.

예시:

```txt
私はりんごを食べます。
```

출력:

```txt
私: 주어
りんご: 목적어
食べます: 동사
```

### 기능 의미

문장 구조 분석 기능은 일본어 문장의 핵심 구성 요소를 구분하여 보여준다. 이를 통해 사용자는 일본어의 기본 어순과 문장 구성 방식을 시각적으로 이해할 수 있다.

---

## 21. 예문 생성 기능

검색한 단어를 활용한 예문을 제공하는 기능이다.

### 기능 의미

예문 생성 기능은 단어가 실제 문장에서 어떻게 사용되는지 보여준다. 이를 통해 사용자는 단어의 뜻뿐만 아니라 문맥 속 활용 방식까지 학습할 수 있다.

---

## 22. 즐겨찾기 기능

자주 사용하는 단어와 문장을 저장하는 기능이다.

### 기능 의미

즐겨찾기 기능은 사용자가 자주 확인하고 싶은 단어, 한자, 문장을 저장할 수 있도록 한다. 이를 통해 반복적으로 확인해야 하는 학습 자료를 효율적으로 관리할 수 있다.

---

## 23. 검색 기록 기능

이전에 검색하거나 번역한 내용을 기록하는 기능이다.

### 기능 의미

검색 기록 기능은 사용자가 이전에 번역하거나 검색한 내용을 다시 확인할 수 있도록 한다. 이를 통해 학습 과정을 누적하고, 이전에 학습한 내용을 반복해서 복습할 수 있다.

---

## 24. 학습 통계 기능

학습 데이터를 시각적으로 보여주는 기능이다.

예시:

```txt
오늘 번역한 문장: 12개
오늘 검색한 한자: 8개
퀴즈 정답률: 76%
```

### 기능 의미

## 28. 결과 통합 화면

번역 결과와 분석 결과를 한 번에 보여주는 기능이다.

예시:

```txt
원문:
私は図書館で日本語を勉強します。

번역:
나는 도서관에서 일본어를 공부합니다。

단어 분석:
私 / は / 図書館 / で / 日本語 / を / 勉強します

한자:
私, 図, 書, 館, 日, 本, 語, 勉, 強

후리가나:
わたしは としょかんで にほんごを べんきょうします。
```

### 기능 의미

결과 통합 화면은 번역, 단어 분석, 한자 추출, 후리가나 표시, 품사 분석 결과를 한 번에 제공한다. 이를 통해 사용자는 일본어 문장을 여러 관점에서 분석하며 종합적으로 학습할 수 있다.

# 앱의 핵심 흐름

1. 사용자는 일본어를 입력한다.
2. 입력 방식은 직접 입력, 음성 입력, 이미지 업로드, 카메라 촬영 중 하나를 선택한다.
3. 앱은 일본어 텍스트를 추출한다.
4. 텍스트를 번역한다.
5. 동시에 단어를 분리하고, 한자를 추출하고, 후리가나를 표시한다.
6. 사용자는 모르는 단어나 한자를 클릭하여 뜻과 읽는 법을 확인한다.
7. 필요하면 단어장에 저장한다.
8. 저장된 단어를 퀴즈로 복습한다.

# 세특용 정리 문장

일본어 학습 과정에서 한자 읽기와 문장 구조 파악에 어려움을 느끼고, 이를 해결하기 위해 일본어 학습 보조 웹 플랫폼을 기획함. 단순 번역 기능을 넘어 한자 자동 추출, 후리가나 표시, 음성 인식, OCR 기반 이미지 문자 인식, 단어 분석 기능 등을 설계하여 일본어의 문자 체계와 문법 구조를 학습할 수 있도록 구현 방향을 설정함. 특히 일본어의 히라가나, 가타카나, 한자가 혼합된 특징에 주목하여 학습자가 문장을 다각도로 분석할 수 있는 기능을 구상함.
